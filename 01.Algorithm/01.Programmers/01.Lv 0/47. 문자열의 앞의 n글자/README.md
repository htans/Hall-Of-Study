# [level 0] 문자열의 앞의 n글자

* [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/181907?language=java)


### 실행문
```java
class Solution {
    public String solution(String my_string, int n) {
        String answer = "";
        
        answer = (my_string.substring(0, n));
        
        return answer;
    }
}
```
