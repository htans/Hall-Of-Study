# [level 0] n 번째 원소까지

* [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/181889)


### 실행문
```java
class Solution {
    public int[] solution(int[] num_list, int n) {
        
        int[] answer = new int[n];
        
        for(int i = 0; i < n; i++ ) {
            answer[i] = num_list[i];
        }
        
        return answer;
    }
}
```
