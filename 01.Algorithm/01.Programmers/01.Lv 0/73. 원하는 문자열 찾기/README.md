# [level 0] 원하는 문자열 찾기

* [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/181878)


### 실행문
```java
class Solution {
    public int solution(String myString, String pat) {

        int answer = 0;
        if(myString.toLowerCase().contains(pat.toLowerCase())){
            answer = 1;
        }
        
        return answer;
    }
}
```
