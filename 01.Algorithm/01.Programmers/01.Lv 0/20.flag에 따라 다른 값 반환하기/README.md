# [level 0] Flag에 따라 다른 값 반환하기

* [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/181933)


### 실행문

```java
class Solution {
    public int solution(int a, int b, boolean flag) {
        int answer = 0;
        
        if(flag == true) {
            answer = a + b;
        } else {
            answer = a - b;
        }
        
        return answer;
    }
}
```
