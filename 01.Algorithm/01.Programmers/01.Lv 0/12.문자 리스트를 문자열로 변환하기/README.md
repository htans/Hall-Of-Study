# [level 0] 문자 리스트를 문자열로 변환하기

* [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/181941)


### 실행문

```java
class Solution {
    public String solution(String[] arr) {
        String answer = "";
        
        for(int i = 0; i < arr.length; i++) {
            answer += arr[i];
        }
        
        return answer;
    }
}
```
