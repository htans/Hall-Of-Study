# [level 0] n의 배수

* [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/181937)


### 실행문

```java
class Solution {
    public int solution(int num, int n) {
        int answer = 0;
        
        if(num % n == 0 ) {
            answer = 1;
        } else {
            answer = 0;
        }
    
        return answer;
    }
}
```
