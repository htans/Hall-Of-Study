# [level 0] 공배수

* [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/181936)


### 실행문

```java
class Solution {
    public int solution(int number, int n, int m) {
        int answer = 0;
        
        if(number % n == 0 && number % m == 0) {
            answer = 1;
        } else {
            answer = 0;
        }
            
        return answer;
    }
}
```
